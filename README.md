# Chikitsa: Clinical Suite

Chikitsa is an advanced, privacy-first clinical tracking and diagnostics platform designed to elevate your healthcare management and clinical scheduling experience.

## Features

- **Symptom Tracker & AI Insights**: Evaluate your symptoms in natural language, powered by cutting-edge AI models.
- **Pharma Plan**: Prescription scheduling and clinical adherence monitoring with precise time vectors.
- **Clinical Journal**: Document your journey, feelings, and symptom chronology locally.
- **Enterprise Grade Security**: Secured using JWT tokens, industry-standard encryption, and a robust local MySQL persistence.

## Getting Started

### Prerequisites

- Node.js (v16 or above)
- MySQL Database Instance

### Setup & Run

1. Clone the repository and navigate to the project directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file from `.env.example` and populate your API keys and MySQL connection variables.
4. Launch the application:
   ```bash
   npm run dev
   ```
